# Tlauncher-for-linux

## install openjdk
### For Ubuntu, Debian, and more(apt)
sudo apt-get install openjdk-8-jre

### For Arch, Manjaro, and more(pacman)
sudo pacman -S jdk-openjdk

### For Fedora, Oracle, RedHat, and more(dnf)
su -c "yum install java-1.8.0-openjdk"

*******************************************

## Run Setupscript 
./setup.sh

## How To play
./tlauncher.sh
